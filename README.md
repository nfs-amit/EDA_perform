# 📊 Exploratory Data Analysis (EDA) Portfolio

Welcome to the **EDA_perform** repository! This project showcases comprehensive Exploratory Data Analysis performed on two distinct real-world datasets: Medical Insurance Costs and Heart Disease records. 

The primary focus of this repository is to demonstrate rigorous **Data / Analytics** workflows, clean raw data, and extract actionable insights that would be valuable for strategic decision-making and **Consulting**.

## 📂 Repository Structure

- `insurance.csv` & `insurance.ipynb` — **Medical Insurance Analysis**
- `heart.csv` & `heart.ipynb` — **Heart Disease Analysis**

## 🎯 Objectives

- **Data Cleaning & Preprocessing:** Handling missing values, formatting data types, and preparing datasets for deeper analysis.
- **Statistical Analysis:** Understanding distributions, variances, and correlations within the data.
- **Data Visualization:** Creating clear, compelling visual representations of the data to communicate findings effectively.
- **Insight Generation:** Translating raw data into strategic insights to solve real-world problems.

## 🔍 Projects Overview

### 1. Medical Insurance Cost Analysis (`insurance.ipynb`)
**Dataset:** `insurance.csv`  
This analysis explores the factors that influence the medical insurance charges billed by health insurance companies. 
* **Key Focus Areas:** * Analyzing the impact of features like age, BMI, and smoking status on insurance charges.
  * Identifying demographic trends and cost drivers to aid in financial forecasting and risk assessment.

### 2. Heart Disease Analysis (`heart.ipynb`)
**Dataset:** `heart.csv`  
This project dives into clinical parameters to identify key indicators and risk factors of heart disease.
* **Key Focus Areas:** * Investigating the relationship between age, cholesterol levels, maximum heart rate, and heart disease presence.
  * Finding patterns that could assist in predictive modeling, early detection, and preventative care planning.

## 🛠️ Technologies Used
* **Python** (Pandas, NumPy for data manipulation)
* **Jupyter Notebook** (Interactive coding and documentation)
* **Matplotlib & Seaborn** (Data visualization)
